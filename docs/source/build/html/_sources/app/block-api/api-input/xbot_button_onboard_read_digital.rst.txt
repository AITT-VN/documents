btn_onboard.is_pressed()
==========

.. image:: images/btn-onboard-1.png
    :scale: 100 %
    :align: center

Lấy giá trị hiện tại của nút nhấn trên board.

Kết quả trả về là ``True`` khi nút được nhấn, hoặc là ``False`` khi nút không được nhấn.
