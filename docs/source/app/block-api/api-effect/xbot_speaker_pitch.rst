speaker.pitch(frequency, time)
==========

.. image:: images/speaker-4.png
    :scale: 100 %
    :align: center

Phát 1 âm thanh với tần số và độ dài truyền vào, trong đó:
    
    - *frequency* : Dữ liệu số, tần số âm thanh được phát và phạm vi giá trị của nó là ``0 ~ 5000``.
    - *time*: Tính theo ``giây``.
